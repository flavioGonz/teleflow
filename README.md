# 🚀 TeleFlow | Infratec Next-Gen PBX

Interfaz de gestión telefónica profesional orientada a Call Centers y Porterías Inteligentes.

## 🛠️ Stack Tecnológico
- **Frontend:** React JS 18 + Tailwind CSS + React Flow.
- **Backend:** PHP API Bridge (Conexión Asterisk & MySQL).

## 📋 Funcionalidades de Gestión de Extensiones (ABM Pro)
- [x] **Buscador Inteligente:** Filtrado instantáneo por nombre, interno o IP.
- [x] **Creador de Extensiones:** Botón rápido para nuevos internos.
- [x] **Edición vía Side Drawer:** Configuración técnica (Nombre, Password, Video, DTMF).
- [x] **Gestión de Avatares:** Subida y visualización de fotos por interno.
- [x] **Monitor de Red:** Visibilidad de IP Origen, MAC y Latencia (RTT).
- [x] **Iconografía Dinámica:** Identificación visual de dispositivos (Deskphone/Softphone).

## 🎧 Módulo de CallCenter (Supervisor)
- [x] **Panel de Colas:** Estado de espera y estrategias de timbrado.
- [x] **Llamadas Vivas:** Monitor de conversaciones en tiempo real.
- [x] **Interacción de Llamadas:** Funciones para colgar o tomar (Pickup) llamadas en curso.
